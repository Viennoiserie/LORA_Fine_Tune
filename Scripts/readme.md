# Setup before training

1. *git clone https://github.com/huggingface/diffusers*

2. *cd diffusers*


3. *pip install -e .*


4. *cd lora_fine_tune/script*

5. *pip install -r requirements_sdxl.txt*


6. *huggingface-cli login*
